Makefile 教程

Linux的make程序用来自动化编译大型源码。make能自动化完成这些工作，是因为项目提供了一个Makefile文件，它负责告诉make，如何编译和链接程序。
make虽然最初是针对C语言开发，但它实际上并不限定于C语言，而是可以应用到任意项目，甚至不是编程语言。

Makefile的安装

自Ubuntu中，可以直接使用apt安装

$ sudo apt install build-essential

安装完成后，可以输入make -v验证：

$ make -v

GNU Make 4.3

Makefile基础

在Linux环境下，当我们输入make命令时，它就在当前目录查找一个名为Makefile的文件，然后，根据这个文件定义的规则，自动化地执行任意命令，包括编译命令。
